# mavisczx.github.io

Personal portfolio website built with React and Material UI.

**Live:** [mavisczx.github.io](https://mavisczx.github.io)

## Tech Stack

- React 18
- Material UI 5
- CSS animations & Intersection Observer
- GitHub Pages

## Development

```bash
npm install
npm start
```

## Deploy

```bash
npm run deploy
```
